# assetManager/client

## 환경설정

1. `$git clone` 한다
2. npm을 초기화한다
    * `$npm install`
3. react 기동하기
    * `npm run start` 맞겠지...? @flowertaekk-dev 는 yarn을 쓰기 때문에 까먹었다.
    * `yarn start`

## IDE 설정

### VS Code

#### Trailing Whitespace 제거하기 (공백 제거하기)

1. File - Preferences - Settings 들어가기 ( `Ctrl` + `,` )
2. `Trim Trailing Whitespace` 검색하기
3. 체크하기!
